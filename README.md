# GENERATIVE-AI2-project
🤖 AI Code Reviewer & Converter

An interactive Streamlit-based web app that uses Google Gemini AI to review, debug, optimize, and convert code across programming languages.

🚀 Features
🔍 Code Review
Get feedback on code quality
Best practices suggestions
Performance improvements
Security insights
🐞 Debug Code
Detect errors in code
Clear explanation of issues
Provides corrected version
⚡ Optimize Code
Improve performance
Enhance readability
Cleaner and efficient code output
🔄 Convert Language
Convert code into another programming language
Preserves logic
Adds helpful comments
🖥️ Tech Stack
Frontend & UI: Streamlit
AI Model: Google Gemini (gemini-2.5-flash)
Language: Python
